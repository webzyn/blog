# 2.2 block 相当于 vue 中的 template

  • <block/> 并不是一个组件，它仅仅是一个包装元素，不会在页面中做任何渲染，只接受控制属性。
<block wx:if="{{type === '2'}}">
  <view>1</view>
  <view>2</view>
</block>

# 4. 模板与引用

# 4.1 模板

https://developers.weixin.qq.com/miniprogram/dev/reference/wxml/template.html (opens new window)

  • WXML 提供模板（template），可以在模板中定义代码片段，然后在不同的地方调用
  • 模板拥有自己的作用域，只能使用 data 传入的数据以及模板定义文件中定义的 <wxs /> 模块
<template name="test">
  <view>test</view>
</template>

<!-- 将模板内容渲染两次 -->
<template is="test"></template>
<template is="test"></template>
