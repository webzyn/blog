# 1

## 小程序

### 1.小程序启动过程

- ① 把小程序的代码包下载到本地
- ② 解析 app.json 全局配置文件
- ③ 执行 app.js 小程序入口文件，<span style='color: red'>调用 App()创建小程序实例</span>
- ④ 渲染小程序首页
- ⑤ 小程序启动完成

### 2.页面渲染的过程

- ① 加载解析页面的.json 配置文件
- ② 加载页面的.wxml 模板和.wxss 样式
- ③ 执行页面的.js 文件，<span style='color: red'>调用 Page()创建页面实例</span>
- ④ 页面渲染完成
